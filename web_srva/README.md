# PHP to HTML and JavaScript Conversion Project

This project aims to convert PHP files into static HTML files and replace PHP code with JavaScript functionality. The goal is to create a fully functional web application using HTML, CSS, and JavaScript.

## Project Structure

```
php-to-html-js
├── src
│   ├── pages
│   │   ├── index.html        # Homepage of the application
│   │   ├── about.html        # About page of the application
│   │   └── contact.html      # Contact page of the application
│   ├── partials
│   │   ├── header.html       # Header section included in other HTML files
│   │   └── footer.html       # Footer section included in other HTML files
│   ├── js
│   │   ├── main.js           # Main JavaScript code for core functionality
│   │   ├── utils.js          # Utility functions for the application
│   │   └── api.js            # Functions for making API calls
│   └── css
│       └── styles.css        # CSS styles for the application
├── package.json               # Configuration file for npm
├── .gitignore                 # Files and directories to be ignored by version control
└── README.md                  # Documentation for the project
```

## Getting Started

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm.

## Usage

- Open the `index.html` file in your web browser to view the homepage.
- Navigate to the `about.html` and `contact.html` files for additional information.
